termaUI v0.1.0

Tibetan typography for the web — authentic fonts, rendering fixes, and visual effects in pure CSS.

Welcome! termaUI gives your website authentic Tibetan fonts, correct Unicode line-breaking, and beautiful text effects — with just two HTML lines and a handful of CSS classes. No fonts to install — every font is bundled in the package and served automatically from the CDN. Use the playground below to try fonts and styles, then copy the generated code straight into your project.
